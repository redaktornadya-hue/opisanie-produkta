# Слайд №62. Практика: пример правильного ответа

**Тип:** белый фон \+ текст (сделать внутри фрейм, разметить текст, добавить скролл, чтобы уместить весь текст)  
**Заголовок:** Пример правильного ответа  
**Текст:**   
**1\. JTBD:** «Когда у меня возник вопрос по продукту в нерабочее время, я хочу быстро получить ответ, чтобы не ждать до утра и не искать информацию самостоятельно».  
**2\. Функции MVP:**

* Функция 1 — Пользователь может задать вопрос боту в Telegram и получить ответ из базы знаний, чтобы решить проблему без ожидания.  
* Функция 2 — Пользователь может выбрать тему вопроса из меню, чтобы бот быстрее нашел нужный ответ.  
* Функция 3 — Если бот не знает ответа, пользователь может оставить заявку, и ему ответят утром.

**3\. Ключевая метрика** — % ответов без участия человека.  
**4\. Гипотезы:**

* Гипотеза 1 — Большинство вопросов пользователей типичные и повторяются. Например, «Как оформить возврат?», «Какие способы оплаты?».  
* Гипотеза 2 — Пользователи готовы общаться с ботом, если он отвечает быстро и по делу.

**5\. Стек инструментов:**

* Telegram Bot API — для создания бота.  
* Claude Code / DeepSeek — для генерации кода и интеграции ИИ для ответов.  
* Google Docs, Microsoft Word, Pages или другой текстовый редактор — для создания базы знаний в формате FAQ.   
* Timeweb, REG.RU или другой сервер — для деплоя.
